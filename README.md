# Learn Go With Tests Acceptance test http & cli application
https://quii.gitbook.io/learn-go-with-tests/build-an-application/app-intro

Following from the learn-go-with-tests book, this repo built an application that can be accessed from the web & cli and created 100% coverage acceptence & unit tests to understand the entire Go-lang testing framework
The application is built into docker containers and the go-lang testcontainers library is used for testing
